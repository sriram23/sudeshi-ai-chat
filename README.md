# Sudeshi AI Chat

AI-powered chat application with real-time streaming, built using Sarvam AI.

## Features
- Streaming responses
- Abortable requests
- Context-aware conversations
- Retry & error handling
- Model switching (Sarvam 30B / 105B)

## Tech Stack
- Next.js (App Router)
- Zustand
- Tailwind CSS
- Sarvam AI
